# SwitchHub – Dépôt officiel pour addons Kodi

Ce dépôt regroupe les addons SwitchHub destinés à améliorer, automatiser et unifier l’expérience Kodi.  
Il contient actuellement :

### Addon disponible
- **plugin.program.kodi.migrate**  
  Outil de migration, sauvegarde et restauration pour environnements Kodi.

### Addons à venir
- Modules pour skins divers
- Fuse2 SkinShortcuts
- Skin général basé sur SkinShortcuts
- Module d’injection pour retour/stop vidéo et retour automatique à l’accueil

---

## 📦 Structure du dépôt
- `/repo/` → Contient les addons prêts à l’installation (ZIP ou dossiers complets)
- **Releases** → Versions packagées pour installation directe via Kodi

---

## 🔧 Fonctionnement général des addons SwitchHub
Les modules SwitchHub offrent :
- Gestion multi-profils
- Sauvegardes et restaurations complètes
- Injection de menus personnalisés
- Automatisation de tâches Kodi
- Architecture modulaire premium
- Compatibilité multi-skins (SkinShortcuts, Fuse2, etc.)

---

## 🚀 Installation
1. Télécharger le ZIP dans la section **Releases**
2. Installer via Kodi :  
   *Addons → Installer depuis un ZIP*

---

## 🛠️ Développement
Les addons SwitchHub sont conçus pour :
- Une architecture indépendante (chaque module est autonome)
- Un code Python propre, modulaire et maintenable
- Une distribution utilisateur (aucune détection automatique de skin)
- Une compatibilité maximale avec les environnements Kodi modernes
