# SwitchHub – Dépôt officiel pour addons Kodi

Ce dépôt contient les modules SwitchHub destinés aux addons Kodi :
- Skins divers
- Fuse2
- SkinShortcuts

## 📦 Structure du dépôt
- `/repo/` → Contient les addons prêts à l’installation
- Releases → Versions packagées (ZIP) pour installation directe

## 🔧 Fonctionnement
Les addons SwitchHub permettent :
- Gestion multi-profils
- Sauvegardes/restaurations
- Injection de menus
- Mises à jour automatiques
- Architecture modulaire premium

## 🚀 Installation
1. Télécharger le ZIP dans la section **Releases**
2. Installer via Kodi :  
   *Addons → Installer depuis un ZIP*

## 🛠️ Développement
Les modules sont conçus pour :
- Architecture indépendante
- Code Python propre, modulaire
- Distribution utilisateur (pas de détection automatique de skin)
